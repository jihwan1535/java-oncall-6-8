1. 비상 근무 배정일 입력
    * 5,월 - 달,요일
    * 잘못된 입력 시 예외 후 재입력
2. 평일 비상 근무 및 휴일 비상 근무 입력
    * 사람 이름
    * 잘못된 입력 시 평일 비상 근무부터 재입력
3. 입력 받은 달을 출력
    * 2월은 28일까지 있음
    * 평일이면서 법정 공휴일 일 때만 (휴일) 표시

근무자

* 근무자는 중복 될 수 없다.
    * 평일, 휴일 각 각 따로 처리 필요
* 이름은 최대 5자이다.
* 근무자는 최소 5명이고 최대 35명이다.

근무자 편성

* 연속 2일은 근무 할 수 없다.
    * 다음 근무자와 교대

공휴일

* 1/1, 3/1, 5/5, 6/6, 8/15, 10/3, 10/9, 12/25
